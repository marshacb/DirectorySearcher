***********File Search Program Notes************

OVERVIEW
--This program accepts a file or directory path along with a list of available actions for each

*******************************************
Run from main in FileSarch.java using command line


***********************************************************************
PROGRAM ACTIONS

-Entering a file path will give you 4 options:

1. Full file information
2. Get the file size only
3. Get the last modified time only
4. Count the lines in a file

Entering a directory will give 2 options:

1. List the full directory and write list to file
2. Output the average size for all files in the given directory

Enter 0 to exit action prompt

*************************************************************************

EXTENSIBILITY

Given public interface, method implementation can be extended for further file processing actions
